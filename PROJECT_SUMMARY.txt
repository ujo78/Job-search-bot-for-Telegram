WHAT YOU HAVE
================================================================================

A fully-functional job search automation system with:

✅ Resume Processing
   • Parse PDF resumes
   • Extract skills, experience, education
   • Compute semantic embeddings

✅ Job Scraping (8+ Job Boards)
   • Indeed, LinkedIn, Glassdoor, Naukri (via JobSpy)
   • RemoteOK, Adzuna, Jobicy (free APIs)
   • Deduplication by URL + fuzzy title/company match

✅ Intelligent Matching
   • Stage 1: SBERT embeddings (free, local)
   • Stage 2: Claude Haiku scoring (0.5¢/search)
   • Stage 3: Manual review (you decide)

✅ Auto-Apply with Preview
   • Detect ATS type (Greenhouse, Lever, Ashby)
   • Pre-fill form fields
   • Screenshot preview
   • User confirmation before submit

✅ Telegram Control
   • /start, /resume, /search, /status, /history
   • Apply/Skip/Save buttons
   • Real-time job digests

✅ 24/7 Automation
   • Scheduled searches every 6 hours
   • Runs on cloud (Railway, Heroku, DigitalOcean)
   • Database persistence
